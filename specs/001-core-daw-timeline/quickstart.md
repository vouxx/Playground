# Quickstart Validation: Core DAW Timeline

Phase 1 구현 후 수동 검증 시나리오.

## 시나리오 1: 첫 트랙 생성 및 재생

1. 앱을 브라우저에서 연다 (`http://localhost:3000`)
2. 빈 타임라인이 마디/박자 그리드와 함께 표시되는지 확인
3. "트랙 추가" 버튼을 클릭
4. **검증**: 새 트랙 행이 타임라인에 나타나고, 트랙 헤더에 이름/볼륨/팬/뮤트/솔로가 표시됨
5. 재생 버튼(또는 Space)을 누름
6. **검증**: 재생 헤드가 왼쪽→오른쪽으로 BPM(120)에 맞춰 이동, 위치 표시기(마디:박자:틱)가 실시간 갱신
7. 정지 버튼(또는 Space)을 누름
8. **검증**: 재생 헤드가 현재 위치에서 멈춤
9. 정지를 다시 누름
10. **검증**: 재생 헤드가 1:1:000으로 이동

## 시나리오 2: 리전 생성 및 편집

1. 트랙이 1개 이상 존재하는 상태에서 시작
2. 타임라인의 빈 영역(마디 3 부근)을 더블클릭
3. **검증**: 4마디 길이의 리전이 생성되고 트랙 색상으로 표시됨
4. 리전을 드래그하여 마디 5로 이동
5. **검증**: 리전이 마디 단위 스냅에 맞춰 마디 5 위치로 이동
6. 리전의 우측 끝을 드래그하여 8마디로 확장
7. **검증**: 리전 길이가 마디 단위로 늘어남
8. 리전을 선택 후 Delete 키
9. **검증**: 리전이 삭제됨

## 시나리오 3: 줌 및 스크롤

1. 타임라인 위에서 마우스 휠 스크롤 (위/아래)
2. **검증**: 타임라인이 시간 축(가로)으로 스크롤
3. Cmd + 마우스 휠로 줌 인
4. **검증**: 그리드 눈금이 더 세밀해지고 리전이 더 크게 표시됨
5. Cmd + 마우스 휠로 줌 아웃
6. **검증**: 그리드가 마디 단위로 돌아가고 더 넓은 시간 범위가 표시됨

## 시나리오 4: 믹서 컨트롤

1. 트랙 2개를 추가하고 각각에 리전 배치
2. 재생 시작 (메트로놈 ON)
3. 트랙 1의 볼륨 슬라이더를 -12dB로 내림
4. **검증**: 해당 트랙의 메트로놈 소리가 작아짐 (Phase 1에서는 메트로놈만 있으므로 시각적 미터로 확인)
5. 트랙 1의 뮤트 버튼 클릭
6. **검증**: 뮤트 시각 표시, 트랙 비활성화 상태
7. 트랙 2의 솔로 버튼 클릭
8. **검증**: 솔로 시각 표시, 트랙 2만 활성 상태

## 시나리오 5: 트랙 관리

1. 트랙 헤더를 우클릭
2. **검증**: 컨텍스트 메뉴(이름 변경/복제/삭제/색상 변경) 표시
3. "이름 변경" → "Drums" 입력
4. **검증**: 트랙 헤더에 "Drums" 표시
5. 트랙 헤더를 드래그하여 다른 트랙 아래로 이동
6. **검증**: 트랙 순서 변경됨

## 성능 검증

1. 트랙 50개 추가, 각 트랙에 리전 20개씩 배치 (스크립트 또는 수동)
2. 줌 인/아웃, 스크롤 조작
3. **검증**: 30fps 이상 유지 (개발자 도구 Performance 탭)
4. 재생 시작 후 재생 헤드 이동
5. **검증**: 끊김 없이 부드러운 이동
